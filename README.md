# react-to_do_list

## Node JS version
v15.13.0 LTS
```
nvm use 15
```

## Make Project
```
npm install -g yarn
yarn add create-react-app
create-react-app .
```
### add library
```
yarn add node-sass@5 classname react-icons
```
- NodeJS 15버전 이상은 node-sass 버전 4 또는 5를 사용해야 하는 것으로 보임 따라서 node-sass@5를 사용하여 5버전의 node-sass를 설치

## Install
```
yarn install
```

## Semantic design
- TodoTemplate
- TodoInsert
- TodoListItem
- TodoList
